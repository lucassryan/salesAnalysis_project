# Análise de Vendas com SQL

Mini projeto de análise de vendas utilizando SQL.

### Objetivo
- Praticar JOINs
- Praticar CTEs
- Realizar agregações com GROUP BY

## 📊 10 perguntas do mini projeto SQL
- 1️⃣ Quantos pedidos existem na base?
- 2️⃣ Quantos clientes existem por estado?
- 3️⃣ Quantos pedidos cada cliente fez?
- 4️⃣ Qual o valor total vendido?
- 5️⃣ Quantos produtos foram vendidos por categoria?
- 6️⃣ Quais são os 10 pedidos mais caros?
- 7️⃣ Qual estado gera mais receita?
- 8️⃣ Quais clientes gastaram mais que a média?
- 9️⃣ Quem são os 5 clientes que mais gastaram? (CTE)
- 🔟 Qual categoria gera mais receita?

##Aprendizado final
SELECT - COUNT - SUM - GROUP BY - ORDER BY - JOIN (simples e múltiplo) - HAVING - SUBQUERY - CTE
